엑셀 파일 업로드 및 다운로드

학과: [시트에 적힌 학과명] → [실제 학부]를 선택한 뒤, 같은 파일을 다시 업로드하세요.

수료 학기: 이름·닉네임으로 행을 찾아, 시트 11열 값에 대응하는 yy-s(예: 25-1)를 입력한 뒤 같은 파일을 다시 업로드하세요.

가입일: 시트·가입일 셀 표시값 기준으로 행을 찾아 yyyy-MM-dd로 입력한 뒤 같은 파일을 다시 업로드하세요.

비액티브 활동학기: DB 학기(yy-s)로 해석되지 않는 칸은 아래에 yy-s로 매핑한 뒤 같은 파일을 다시 업로드하세요.

비액티브 예정복귀: DB 학기(yy-s)로 해석되지 않는 칸은 아래에 yy-s로 매핑하세요. 매핑 입력을 비우면 해당 표시값은 학기로 쓰지 않습니다(신규 비액티브는 서버 기본 규칙 적용, 이미 비액티브인 멤버는 기존 예정복귀 유지).
